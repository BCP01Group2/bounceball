## 기본조작법(키보드조작법)
> 숫자키를 사용하여 볼의 기능을 사용할 수 있습니다.
>
> 1번 키 : 노란색 (일반 점프)
>
> 2번 키 : 파란색 (2단뛰기)
>
> 3번 키 : 빨간색 (오른쪽방향으로 나아가기)
>
> 4번 키 : 보라색 (왼쪽방향으로 나아가기)
>
> 이동은 방향키를 이용하여 조작합니다.

세 단계를 모두 excellent로 클리어하면 숨겨진 맵을 볼 수 있습니다!
